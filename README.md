# Calinender - Menstruatiekalender

Een privacyvriendelijke menstruatiekalender app. Alle data blijft lokaal op je apparaat.

## Live Demo

**https://calinender.vercel.app/**

## Features

- Kalender met periode tracking
- Symptomen registratie (stemming, pijn, energie)
- Voedingsadvies per cyclusfase
- Bewegingsadvies per cyclusfase
- Recepten afgestemd op je cyclus
- Volledig priv√© - alle data blijft lokaal

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Radix UI
- Local Storage voor data persistence

## Development

```bash
npm install
npm run dev
```
